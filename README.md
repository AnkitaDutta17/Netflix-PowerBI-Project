# 📊 Netflix Data Analysis Dashboard (Power BI) 

## 🔹 Project Overview
This project presents an interactive Power BI dashboard built to analyze Netflix content data. The dashboard provides insights into content distribution, trends over time, and key patterns across genres, countries, and ratings.
 

## 🔹 Objectives
- Analyze the distribution of Movies vs TV Shows  
- Identify trends in content addition over the years  
- Explore content by genre, country, and rating  
- Derive insights to understand Netflix’s content strategy  


## 🔹 Tools & Technologies
- Power BI  
- Data Cleaning & Transformation  
- Data Visualization  


## 🔹 Dashboard Features
- Interactive filters (Year, Genre, Country, Type)  
- KPI cards for quick insights  
- Trend analysis over time  
- Distribution analysis by category  


## 🔹 Key Insights
- Significant growth in content addition after 2015  
- Movies dominate the platform compared to TV Shows  
- Content is concentrated in a few key countries  
- Popular genres show consistent trends over time  


## 🔹 Files in Repository
- `Netflix-PowerBI-Project.pbix` → Power BI dashboard  
- `Netflix TV Shows and Movies.xlsx` → Dataset used
